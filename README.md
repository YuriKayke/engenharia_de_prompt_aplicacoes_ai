------- Engenharia de Software -------

A disciplina de Engenharia de Software estuda os princípios, métodos e ferramentas utilizados para o desenvolvimento de sistemas de software de forma organizada, eficiente e com qualidade. Seu foco é ensinar como planejar, construir, testar e manter aplicações, considerando aspectos técnicos e também de gestão de projetos.

Durante a disciplina, são abordados temas como levantamento e análise de requisitos, modelagem de sistemas, metodologias de desenvolvimento (como ágil e cascata), controle de versões, testes de software e boas práticas de programação. Além disso, também se discute a importância da colaboração em equipe, documentação e manutenção contínua do software.

O objetivo principal da disciplina é preparar os estudantes para desenvolver soluções confiáveis, escaláveis e alinhadas às necessidades dos usuários e do mercado.
